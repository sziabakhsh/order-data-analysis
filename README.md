# Order Data Analysis Project

## Overview

This project performs exploratory data analysis on an orders dataset using Python.
The goal is to analyze sales performance, identify top customers, and visualize key business insights.

The analysis includes data quality checks, statistical summaries, sales aggregation, and data visualization.

---

## Technologies Used

* Python
* Pandas
* Seaborn
* Matplotlib
* Visual Studio Code
* Git & GitHub

---

## Dataset

The dataset contains simulated order data with approximately 500 records.

Columns include:

* `order_id`
* `date`
* `customer`
* `city`
* `amount`

## Analysis Steps

The project follows a typical data analysis workflow:

1. Load dataset
2. Data quality checks
3. Basic statistical analysis
4. Sales aggregation
5. Advanced analysis
6. Data visualization
7. Export results

## Key Insights

Examples of insights generated by the analysis:

* Identification of top-performing cities

## Visualizations

The project generates visualizations such as:

* Sales by city (bar chart)

* Top customers (bar chart)

---

## How to Run the Project

Clone the repository:

```
git clone https://github.com/sziabakhsh/order-data-analysis.git
```

Navigate to the project folder:

```
cd order-data-analysis
```

Run the analysis script:

```
python analysis.py
```
